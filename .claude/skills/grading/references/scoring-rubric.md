## 1. 첫인상 (5점)

### 구조적 완성도 (2점)

| 점수 | 기준 | 체크리스트 |
|------|------|------------|
| 2.0 | 서론-본론-결론 명확, 목차 논리적, 단락 구분 완벽 | ✓ 서론: 정의 명확<br>✓ 본론: 그림+표<br>✓ 결론: 발전방향<br>✓ 목차만 봐도 이해 가능 |
| 1.5 | 구조는 있으나 일부 불명확하거나 단락 구분 미흡 | ✓ 3단 구조 있음<br>✗ 일부 섹션 불명확 |
| 1.0 | 구조가 약하거나 불완전 | ✓ 구조 시도는 했으나 불완전 |
| 0.5 | 구조가 매우 불명확 | ✗ 구조 거의 없음 |

### 시각적 완성도 (2점)

| 점수 | 기준 | 체크리스트 |
|------|------|------------|
| 2.0 | 다이어그램+표 모두 있고, 깔끔하고 균형잡힌 레이아웃 | ✓ 다이어그램 있음<br>✓ 표 있음<br>✓ 깔끔한 레이아웃<br>✓ 가독성 좋음 |
| 1.5 | 둘 다 있으나 일부 조잡하거나 배치 불균형 | ✓ 다이어그램+표 있음<br>✗ 일부 조잡함 |
| 1.0 | 하나만 있거나 둘 다 조잡함 | ✓ 다이어그램만 또는 표만<br>✗ 품질 낮음 |
| 0.5 | 둘 다 없거나 매우 조잡함 | ✗ 시각 자료 없음 또는 매우 조잡 |

### 분량 (1점)

| 점수 | 기준 |
|------|------|
| 1.0 | 1페이지 분량 적절 (너무 빈약하지도, 과하지도 않음) |
| 0.7 | 약간 부족하거나 약간 과다 |
| 0.5 | 분량 매우 부족 또는 매우 과다 |

## 점수대별 종합 평가

### 25-30점: 우수 (A)

**특징:**
- 모든 평가 기준 충족
- 구조, 내용, 전문성 모두 뛰어남
- 독창적 견해와 통찰력
- 실무 경험 잘 녹아있음

**평가:**
"기술사 수준의 우수한 답안. 구조적 완성도와 전문성이 뛰어나며, 독창적인 견해를 제시함. 실무 경험이 잘 드러나고 최신 트렌드를 반영함."

### 18-24점: 합격 수준 (B)

**특징:**
- 기본 구조 갖춤
- 요구사항 대부분 충족
- 일부 영역 개선 필요
- 전문성은 있으나 깊이 부족

**평가:**
"합격 가능한 수준의 답안. 기본 구조와 내용을 갖추었으나 일부 영역에서 깊이나 통찰력이 부족함. 개선 여지가 있음."

### 12-17점: 보완 필요 (C)

**특징:**
- 구조 불완전
- 요구사항 일부만 충족
- 전문성 부족
- 많은 개선 필요

**평가:**
"추가 학습과 연습이 필요한 수준. 기본 구조는 있으나 불완전하고, 내용의 깊이와 전문성이 부족함. 전면적인 보완 필요."

### 0-11점: 불합격 (D)

**특징:**
- 기본 구조 미흡
- 요구사항 미충족
- 내용 부실
- 전면 재작성 필요

**평가:**
"기본 요건을 충족하지 못한 답안. 구조와 내용 모두 전면 재작성이 필요함. 기본 개념부터 재학습 권장."

## 채점 시 주의사항

### 1. 객관성 유지
- 개인 선호가 아닌 기준에 따라 평가
- 동일한 기준을 모든 답안에 일관되게 적용
- 주관적 판단 최소화

### 2. 증거 기반 채점
- 구체적 근거를 들어 점수 부여
- 채점 근거를 명확히 기록
- 예시를 들어 설명

### 3. 건설적 피드백
- 단순히 점수만 부여하지 않음
- 왜 그 점수를 받았는지 설명
- 어떻게 개선할 수 있는지 구체적 제시

### 4. 균형 잡힌 평가
- 긍정적 측면과 개선 측면 모두 언급
- 잘한 점을 먼저 언급 후 개선점 제시
- 격려와 동기부여 포함

### 5. 맥락 고려
- 문제의 난이도 고려
- 시간 제약 고려
- 신기술 vs 성숙기술 특성 고려

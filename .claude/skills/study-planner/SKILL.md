# Study Planner (학습 계획 수립)

## Overview

Create comprehensive, personalized study plans for IT Professional Examination preparation. Analyzes weak areas, sets priorities, creates timeline-based schedule, recommends resources, and tracks progress.

## Planning Workflow

When user requests a study plan:

1. **Assess current level**
   - Identify strong domains
   - Identify weak domains
   - Evaluate answer writing ability
   - Check knowledge gaps

2. **Set goals and timeline**
   - Target exam date
   - Available study hours per week
   - Target score level
   - Specific improvement goals

3. **Prioritize topics**
   - High-frequency exam topics
   - Weak areas that need improvement
   - Latest trends to cover
   - Foundation vs advanced topics

4. **Create schedule**
   - Week-by-week plan
   - Daily study targets
   - Topic rotation
   - Practice milestones

5. **Recommend resources**
   - Study materials for each topic
   - Practice questions
   - Reference materials
   - Online resources

6. **Set checkpoints**
   - Weekly self-assessment
   - Practice answer writing
   - Mock exam schedule
   - Progress review

## 학습 전략

### 효율적 학습 방법
1. **Active Recall**: 보지 않고 설명해보기
2. **Spaced Repetition**: 일정 간격으로 복습
3. **Interleaving**: 여러 주제 섞어서 학습
4. **Deliberate Practice**: 약점 집중 공략

### 답안 작성 전략
1. **구조 먼저**: 서론-본론-결론 틀 잡기
2. **시간 관리**: 25분 엄수
3. **템플릿 활용**: 문제 유형별 템플릿
4. **피드백 적용**: 리뷰 내용 반영

### 컨디션 관리
- 충분한 수면 (7-8시간)
- 규칙적인 학습 시간
- 적절한 휴식 (50분 학습, 10분 휴식)
- 시험 일주일 전 컨디션 조절

## Adaptive Planning

Study plan adjusts based on:
- **Progress rate**: Speed up or slow down
- **Assessment results**: Focus on weak areas
- **Mock exam scores**: Intensify practice
- **Time remaining**: Adjust depth vs breadth

## Customization Options

- **Intensive mode**: More hours, faster pace
- **Balanced mode**: Regular pace, work-life balance
- **Slow & steady**: Longer timeline, deep learning
- **Domain-focused**: Emphasis on specific areas
- **Practice-heavy**: More answer writing, less theory

## Example Usage

**User request:** "시험이 3개월 남았고, 주당 15시간 공부할 수 있습니다. 클라우드와 보안이 약합니다. 학습 계획 세워주세요."

**Process:**
1. Calculate: 12 weeks, 15 hours/week = 180 hours total
2. Prioritize: Cloud & Security (weak) → 60%
3. Allocate: Other domains → 30%, Practice → 10%
4. Create weekly schedule
5. Set checkpoints and milestones
6. Recommend resources

**Result:** 12-week personalized study plan with weekly goals and resource links

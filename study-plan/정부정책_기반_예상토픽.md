# 정부정책 기반 예상토픽 (2024-2025)

## 📋 개요

**목적**: 정부기관(KISA, NIA, 과기정통부, 행안부)의 2024-2025 IT 정책/사업을 분석하여 138회 이후 출제 예상 토픽 도출

**핵심 인사이트**:
- 정보관리기술사는 **국가사업 제안서/발주 시 기술 자문** 역할
- 정부 정책 방향 = 출제 트렌드
- 과거 패턴: 정책 발표 후 6-12개월 이내 출제

---

## 🏛️ 1. 행안부 (Ministry of Interior and Safety)

### 정책 1: 2025년까지 공공기관 100% 클라우드 전환

**배경**:
- 2025년까지 19,000개 공공 시스템 전면 클라우드 전환
- 2024년 12월 기준 진행 중
- Cloud-First 정책 법제화

#### 예상토픽 1-1: **공공 클라우드 전환 전략 및 마이그레이션 방법론**

**출제 각도**:
> "행정안전부는 2025년까지 전 공공기관의 시스템을 클라우드로 전환하는 정책을 추진 중이다.
> 레거시 시스템을 클라우드로 마이그레이션할 때 고려해야 할 기술적 전략과 단계별 방법론을 설명하시오."

**핵심 키워드**:
- 6R 전략 (Rehost, Replatform, Refactor, Repurchase, Retain, Retire)
- 클라우드 네이티브 아키텍처
- 데이터 마이그레이션 전략
- 하이브리드 클라우드 운영

**관련 기술**:
- IaC (Infrastructure as Code): Terraform, Pulumi
- 컨테이너화 (Docker, Kubernetes)
- 서비스 메시 (Istio, Linkerd)
- 클라우드 비용 최적화 (FinOps)

**참고**: 136회 서버리스 컴퓨팅 출제 → 클라우드 아키텍처 확장 예상

---

#### 예상토픽 1-2: **CSAP (Cloud Security Assurance Program) 인증 요구사항**

**출제 각도**:
> "공공 클라우드 서비스 이용 시 CSAP 인증이 의무화되었다.
> CSAP의 보안 요구사항과 기존 ISMS-P와의 차이점을 설명하시오."

**핵심 키워드**:
- 클라우드 보안 인증 체계
- 공동 책임 모델 (Shared Responsibility Model)
- 멀티 테넌시 보안
- 데이터 주권 (Data Sovereignty)

**관련 기술**:
- CWPP (Cloud Workload Protection Platform)
- CASB (Cloud Access Security Broker)
- CSPM (Cloud Security Posture Management)
- 클라우드 암호화 (KMS, HSM)

### 정책 2: N2SF (National Network Security Framework) 도입

**배경**:
- 기존 물리적 망분리 → 논리적 다계층 보안으로 전환
- C/S/O (Confidential/Sensitive/Open) 3계층 보안 모델
- 업무 효율성 개선 + 보안 강화

#### 예상토픽 1-3: **N2SF 기반 망분리 대체 전략**

**출제 각도**:
> "N2SF(National Network Security Framework)는 기존 물리적 망분리를 논리적 다계층 보안으로 대체한다.
> C/S/O 계층별 보안 요구사항과 구현 방안을 설명하시오."

**핵심 키워드**:
- 마이크로세그멘테이션 (Micro-segmentation)
- 네트워크 접근 제어 (NAC)
- SDP (Software Defined Perimeter)
- ZTNA (Zero Trust Network Access)

**관련 기술**:
- 네트워크 가상화 (SDN, NFV)
- 엔드포인트 보안 (EDR, XDR)
- 데이터 다이오드 (Data Diode)
- VDI (Virtual Desktop Infrastructure)

**참고**: 137회 N2SF 출제 → 138회 **구현 상세** 출제 가능성 높음

## 📝 답안지 작성 우선순위 제안

**Phase 1: 정부 정책 직결 (출제 확률 최고)**
1. N2SF 기반 망분리 대체 전략
2. 버티컬 AI 데이터 구축 전략
3. NIST PQC 표준 알고리즘 구현

4. LLMOps 파이프라인
5. 컨테이너 보안 (eBPF 기반)
6. 초거대 AI 인프라 아키텍처

**Phase 3: 신규 트렌드**
7. AI 신뢰성 확보 기술 (XAI)
8. 프롬프트 인젝션 방어
9. 멀티 클라우드 보안 아키텍처
10. 양자 컴퓨팅 활용 시나리오
